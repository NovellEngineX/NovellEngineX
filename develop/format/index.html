Формат файла сценария Novell Engine X

Ниже указан формат "script.nex". Первая строка с комментарием в файле обязательна.

1 строка - имя говорящего (используйте знак "|" для обозначения второго говорящего: Первый|Второй)

2 строка - текст (используйте переменную "{line}" для переноса строки: Первая строка{line}Вторая строка)

3 строка - название картинки из папки images (можно .gif) (если расширение файла не указано, то движок будет использовать .jpg: test = test.jpg)

Совет: Если Вы хотите оставить предыдущее изображение, то вместо названия файла укажите "$old-img"

Расположения файлов конфигурации Novell Engine X

   [GUI] data\icon.ico - иконка главного окна (файл .ico)

   [GUI] data\gui\next_btn.nex - текст кнопки просмотра

   [GUI] data\gui\theme.nex - тема среды (Светлая/темная)

   [GUI] data\gui\title.nex - заголовок главного окна, название новеллы

   [GUI] data\gui\details.nex - файл с параметрами главного окна

   [SCRIPT] data\story\script.nex - сценарий новеллы в строго заданном формате

   [SCRIPT] data\story\the_end\end.nex - текст, показывающийся после просмотра всего сценария

   [SCRIPT] data\story\the_end\end_name.nex - "имя говорящего" при показе кадра end.jpg

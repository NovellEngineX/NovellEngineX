Формат файла тем Novell Engine X

Ниже указан формат "details.nex". Первая строка с комментарием в файле обязательна.

Для установки темы используйте приставку setTheme=

 Lite - белая тема

 Dark - тёмная тема

 WildRed - "дикий красный", чёрно-красная тема

 Blue - голубая тема

 Ping - розовая тема
